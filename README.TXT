COMPILAR

A la carpeta base del projecte, executar la següent comanda:

$ make

EXECUTAR

A la carpeta base del projecte, executar la següent comanda:

$ ./AirlineScheduling

A continuació apareixerà una descripció de les opcions d'execució del programa. Tornar a executar amb les opcions adequades:

$ ./AirlineScheduling [-12hl] [-a algoritme] [-AM]
